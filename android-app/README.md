# Gemini CLI Mobile

تطبيق Android لـ Gemini CLI - أداة سطر الأوامر المدعومة بالذكاء الاصطناعي من Google.

## الميزات

### الميزات الأساسية
- 🤖 دردشة مع Gemini AI مباشرة من هاتفك
- 📷 رفع ومشاركة الصور مع AI للتحليل والوصف
- 🎤 تسجيل وإرسال الرسائل الصوتية
- 🔊 تشغيل الرسائل الصوتية مع تحكم كامل
- 💬 إدارة محادثات متعددة منفصلة
- 🔒 تخزين آمن لمفتاح API والمحادثات
- 🌙 واجهة مظلمة أنيقة
- 📱 تصميم متجاوب يناسب جميع أحجام الشاشات
- ⚡ استجابة سريعة وأداء محسن
- 🖼️ عرض الصور بملء الشاشة مع تكبير وتصغير
- 📝 توليد عناوين ذكية للمحادثات تلقائياً
- 🗂️ حفظ تاريخ جميع المحادثات محلياً

### الميزات المتقدمة
- 💻 محرر أكواد مدعوم بالذكاء الاصطناعي
- 🌐 بحث على الإنترنت متكامل
- 🛠️ منشئ مشاريع لإنشاء تطبيقات جديدة
- 📦 مساعد إنشاء ملفات APK
- 🎨 تخصيص واجهة المستخدم

### نظام الإشعارات والإعدادات
- 🔔 إشعارات ذكية قابلة للتخصيص
- 🔧 إعدادات متقدمة شاملة لـ AI والواجهة

## التثبيت

### الطريقة الأولى: تحميل APK (قريباً)
سيتم توفير ملف APK جاهز للتحميل والتثبيت مباشرة.

### الطريقة الثانية: البناء من المصدر

1. تأكد من وجود Node.js 18+ على نظامك
2. استنسخ المستودع:
   ```bash
   git clone https://github.com/you112ef/gemini-cli.git
   cd gemini-cli/android-app/GeminiCLI
   ```

3. ثبت التبعيات:
   ```bash
   npm install
   ```

4. شغل التطبيق في وضع التطوير:
   ```bash
   npx expo start
   ```

## الإعداد

1. احصل على مفتاح API من [Google AI Studio](https://aistudio.google.com/apikey)
2. افتح التطبيق وأدخل مفتاح API في شاشة الإعدادات
3. ابدأ المحادثة مع Gemini AI!

## البناء للإنتاج

### بناء APK محلياً
```bash
npx expo build:android
```

### باستخدام EAS Build (موصى به)
```bash
npx eas build --platform android
```

## الاستخدام

1. **إعداد API Key**: عند فتح التطبيق لأول مرة، ستحتاج لإدخال مفتاح Gemini API
2. **بدء المحادثة**: اكتب سؤالك في مربع النص واضغط إرسال
3. **رفع الصور**: 
   - اضغط على أيقونة الكاميرا 📷
   - اختر "من المعرض" أو "التقاط صورة"
   - أرسل الصورة مع أو بدون نص
4. **تسجيل الصوت**:
   - اضغط مطولاً على أيقونة الميكروفون 🎤
   - تحدث وسيتم تسجيل رسالتك الصوتية
   - اترك الزر لإرسال التسجيل
5. **استخدام الأدوات المتقدمة**:
   - **محرر الأكواد** 💻: لتوليد وعرض أكواد البرمجة
   - **البحث على الإنترنت** 🌐: للبحث عن المعلومات
   - **منشئ المشاريع** 🛠️: لإنشاء قوالب تطبيقات جديدة
6. **إدارة المحادثات**:
   - اضغط على أيقونة المحادثة 💬 لعرض جميع المحادثات
   - إنشاء محادثة جديدة
   - حذف المحادثات غير المرغوب فيها
   - التنقل بين المحادثات المختلفة
7. **عرض الوسائط**:
   - اضغط على أي صورة لعرضها بملء الشاشة مع إمكانية التكبير
   - اضغط على الرسائل الصوتية لتشغيلها
8. **الإعدادات المتقدمة**:
   - اضغط على أيقونة الأدوات 🔧 للوصول للإعدادات الشاملة
   - تحكم في إعدادات AI (نموذج، إبداعية، إلخ)
   - إعدادات الواجهة والإشعارات
   - إعدادات الأمان والخصوصية
9. **إحصائيات الاستخدام**:
   - اضغط على أيقونة الإحصائيات 📊 لعرض تفاصيل الاستخدام
   - متابعة عدد الرسائل والصور والصوتيات
   - إحصائيات وقت الاستخدام والنشاط

## الأمان والخصوصية

- يتم تخزين مفتاح API محلياً على الجهاز باستخدام AsyncStorage
- لا يتم إرسال المفتاح إلا إلى خوادم Google المخصصة لـ Gemini API
- جميع المحادثات والصور محفوظة محلياً ولا يتم إرسالها لخوادم خارجية
- الصور يتم تحويلها إلى base64 وإرسالها مباشرة لـ Gemini API
- لا يتم حفظ الصور بشكل دائم، يتم حذفها عند إغلاق التطبيق
- الأذونات المطلوبة:
  - **الكاميرا**: لالتقاط الصور والفيديو
  - **المعرض**: لاختيار الصور الموجودة
  - **الميكروفون**: لتسجيل الرسائل الصوتية
  - **التخزين**: لحفظ الوسائط والمحادثات
  - **الإنترنت**: للتواصل مع Gemini API والبحث على الإنترنت
